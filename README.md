Make sure Node and npm are installed on your machine, for more information visit [this link](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

Clone the repository to local machine

Navigate to the cloned folder from a terminal

run `npm i` and wait for the dependencies to install

run `npm start` and a browser window should open

if it does not then open a browser and navigate to localhost:3000
